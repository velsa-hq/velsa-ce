This release focuses on security and compliance hardening. Most of the
work is behind the scenes - there are no changes to your day-to-day
workflows.

### Improved

- **Tamper-evident audit logging.** Audit events are now streamed to a
  separate, write-protected destination so the activity record can't be
  altered after the fact.
- **Hardened deployment platform.** The web server and application
  container were brought in line with government security technical
  implementation guides (STIG / SRG).

### Fixed

- A range of smaller security and configuration items identified during a
  control-by-control review.
